# MAXEL-OS Performance Monitor
A lightweight resource monitor designed for MAXEL-OS (Fedora-based). 

### Features:
* Real-time CPU & RAM monitoring.
* Universal GPU support (Intel, AMD, NVIDIA).
* One-click System Cache Clearing.
* Dark Mode Professional UI.

### How to run:
1. Give permissions: `chmod +x run.sh ui.py`
2. Run the script: `./run.sh`
